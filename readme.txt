模块介绍：
  java-dao:主要包含实体类，mapper的接口，xml以及自动生成代码的example
  java-dto:主要存放入参和出参，
  java-http:所有http请求的入口；
  java-server:作为被访问的服务器，之前主要是被http请求调用
  java-service:业务模块，实现各种业务逻辑，主要被http、server等部分调用；
  java-springboot-learn:springboot的学习内容
  java-util:工具类模块，包含各种工具类、异常类、静态常量
  java-webservice-server:webservice服务端，写webservice服务端代码，被客户端调用，目前尚未实现
